ခုနစ်နေ့ဘုရားရှိခိုး

သန္ဓေယူသောနေ့ ( မ - သန် )

၁။ သုံးလူ့ရှင်ပင်၊ ကျွန့်ထိပ်တင်၊ သောင်းခွင်စင်္ကြာဝဠာ၊ နတ်ဗြဟ္မာတို့၊ ညီညာရုံးစု၊ တောင်းပန်မှုကြောင့်၊ ရတုနဂို၊ ရွှန်းရွှန်းစိုသည်၊ ဝါဆိုလပြည့်၊ ကြွက်မင်းနေ့ဝယ်၊ ချမ်းမြေ့ကြည်ဖြူ၊ သန္ဓေယူသည် … နတ်လူငြိမ်းဖို့ ကိန်းပါကို၊ ဖွားမြင်တော်မူသောနေ့ ( သိ - ဖွား )

၂။ သန္ဓေယူကာ၊ ဆယ်လကြာသော်၊ မဟာသက္ကရာဇ်၊ ခြောက်ဆဲ့ရှစ်ကြုံ၊ ကဆုန်လပြည့်၊ သောကြာနေ့ဝယ်၊ ချမ်းမြေ့စုံစီ၊ လုမ္ဗဏီ၌၊ မဟီလှိုက်ဆူ၊ ဖွားတော်မူသည် … နတ်လူ အောင်မြို့ လမ်းပါကို၊ တောထွက်တော်မူသောနေ့ (ကြ - တော)

၃။ ဖွားမြင်မြောက်သော်၊ ဆယ့်ခြောက်နှစ်ရွယ်၊ ပျိုနုနယ်၌၊ သုံးသွယ်ရွှေနန်း၊ သိမ်းမြန်းပြီးလစ်၊ ဆယ့်သုံးနှစ်လျှင်၊ ဘုန်းသစ်လျှံလူ၊ စံတော်မူ၍၊ ရွယ်မူနုဖြိုး၊ နှစ်ဆယ့်ကိုးဝယ်၊ လေးမျိုးနိမိတ်၊ နတ်ပြဟိတ်ကြောင့်၊ ရွှေစိတ်ငြင်ငြို၊ သံဝေပိုက၊ ဝါဆိုလပြည့်၊ ကျားမင်းနေ့ဝယ်၊ ချမ်းမြေ့ရဂုံ၊ တောရပ်လှုံသည် … စုံမြိုင်ပင်ရိပ် ခန်းမှာကို။ ဘုရားဖြစ်တော်မူသောနေ့ ( လတ် - ပွင့် )

၄။ တောရပ်မြိုင်ပေါ်၊ ခြောက်နှစ်ပျော်၍၊ ခါတော်တစ်ပုံ၊ ပွင့်ချိန်ကြုံက၊ ကဆုန်လပြည့်၊ ဆင်မင်းနေ့ ဝယ်၊ မြေ့ပရမေ၊ ပလ္လင်ဗွေထက်၊ ရွှေညောင်တော်ကြီး၊ ဗိမာန်ထီးနှင့်၊ မငြီးကြည်ဖြူ၊ နေတော်မူလျက်၊ ရန်မြူခပင်း၊ အမိုက်သင်းကို၊ အရှင်းပယ်ဖျောက်၊ အလင်းပေါက်က၊ ထွန်းတောက်ဘုန်းတော်၊ သောင်းလုံး ကျော်သည် … သုံးဖေါ်လူတို့ ငြမ်းပါကို၊ ဓမ္မစကြာဟောတော်မူသောနေ့ ( တွေး - စက် )

၅။ ဘုရားဖြစ်ခါ၊ မိဂဒါသို့၊ စကြာရွှေဘွား၊ ဖြန့်ချိသွား၍၊ ငါးပါးဝဂ္ဂီ၊ စုံအညီနှင့်၊ မဟီတစ်သောင်း၊ တိုက်အပေါင်းမှ၊ ခညောင်းကပ်လာ၊ နတ်ဗြဟ္မာအား၊ ဝါဆိုလပြည့်၊ စနေနေ့ဝယ်၊ ကြွေ့ကြွေ့လျှံတက်၊ ဓမ္မစက်ကို၊ မိန့်မြွက်ထွေပြား၊ ဟောဖော်ကြားသည် … တရားနတ်စည် ရွမ်းတယ်ကို၊ ပရိနိဗ္ဗာန်စံတော်မူသောနေ့ ( ဆ - စု )

၆။ တရားနတ်စည်၊ ဆော်ရွမ်းလည်က၊ သုံးမည်ဘုံသိုက်၊ တစ်သောင်းတိုက်ဝယ်၊ ကျွတ်ထိုက်သသူ၊ နတ်, လူ, ဗြဟ္မာ၊ သတ္တဝါကို၊ ခေမာသောင်သို့၊ ဖောင်ကူးတို့ဖြင့်၊ ဆောင်ပို့ပြီးခါ၊ ဝါလေးဆယ့်ငါး၊ သက်ကားရှစ်ဆယ်၊ စုံပြည့်ကြွယ်က၊ ရာလေးဆယ့်ရှစ်၊ သက္ကရာဇ်ဝယ်၊ နယ်မလ္လာတိုင်း၊ စံနှိုင်းမယုတ်၊ ကုဿိန္နာရုံ၊ အင်ကြင်းစုံ၌၊ ကဆုန်လပြည့်၊ အင်္ဂါနေ့ဝယ်၊ ချမ်းမြေ့နိဗ္ဗူ၊ စံတော်မူသည် … ဝှန်းဆူသောင်းလုံး ကျော်တယ်ကို ၊ တေဇောဓာတ်လောင်တော်မူသောနေ့ ( အပ် - သင့်)

၇။ နိဗ္ဗူစံပြီ၊ ရှုမအီသား၊ သိင်္ဂီရွှေလှော်၊ အလောင်းတော်ကို၊ ထိုရော်ကဆုန်၊ လဆုတ်ကြုံ၍၊ ဂဠုန်နေ့ဝယ်၊ ခိုးငွေ့မနှော၊ ဓာတ်တေဇောလျှင်၊ ရှင်စောဉာဏ်စက်၊ ဓိဋ္ဌာန်ချက်ဖြင့်၊ လျှံတက်ကော်ရော်၊ မီးပူဇော်သည် … မွေတော်ရှစ်စိတ် ကြွင်းတယ်ကို၊

ဆုတောင်း

ထိုခုနစ်နေ့၊ သာချမ်းမြေ့ကို၊ ဆွေ့ဆွေ့ကြည်ဖြူ၊ အာရုံယူလျက်၊ သုံးလူတို့နတ်၊ မြတ်ထက်မြတ်သား၊ အထွဋ်ခေါင်ဖျား၊ ကိုယ်တော်ဘုရားကို၊ သုံးပါးမှန်စွာ၊ ဝန္ဒနာဖြင့်၊ သဒ္ဓါဖြိုးဖြိုး၊ ကျွန်ရှိခိုးသည် … ကောင်းကျိုးခပ်သိမ်း ကြွယ်စေသောဝ်။
